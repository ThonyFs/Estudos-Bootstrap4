classes de margin & padding :

  • .mt-, mb-, ml-, .mr-: Margin para direções top, bottom, left, right ( valores de 0 a 5)
  • .m- : margin para todas direções
  • .mx- : margin para eixo x (horizontal, direita e esquerda)
  • .my- : margin para eixo x (vertical, top e bottom)
  • .pt-, pb-, pl-, .pr-: padding para direções top, bottom, left, right ( valores de 0 a 5)
  • .p- : Padding para todas direções
  • .px- : Padding para eixo x (horizontal, direita e esquerda)
  • .py- : Padding para eixo x (vertical, top e bottom)
