Class :

  • .float-(medida: sm, md, lg, xl)-left/right : Faz os elementos flutuarem dentro do seu container.
  • .clearfix: Limpa o fluxo de flutuação, utilizar na div pai.
  • .sticky-top/bottom : Cola um elemento no topo ou na base, caso o elemento estiver no centro da viewport, ao tocar o topo/base ele fixará.
  • .fixed-top/botton : Fixa no topo ou base o elemento
  • .list-inline: Deixa a lista posicionada em linha
  • list-inline-item: Utilizado para os itens da lista
