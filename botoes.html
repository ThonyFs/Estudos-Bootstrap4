Readme

Class:

  • .btn : classe padrão para utilização de botão
  • .btn-(cor) : botoões com cores.
  • .btn-outline-(cor) : botoões com bordas da cor especificada
  • .btn-sm/lg : tamanhos dos botõess
  • .btn-block : botão em bloco
  • .btn + active ou disabled : controla o estado do botão
  • data-toggle="button" : Propriedade que alterna entre os estados do botão
  • .btn-group : aplicar a classe em uma div que contém os botões para agrupar
  • .btn-group-vertical : aplicar a classe em uma div que contém os botões para agrupar na vertical
  • .btn-toolbar : Aplicar em uma div que contem mais de um grupo de botões para agrupar formando uma toolbar
  • .dropdown : Aplicar em uma div que contenha 1 botão + div com links para criar um botão com dropdown.
  • .dropdown-toggle + data-toggle="dropdown" : Aplicar no botão que criará o dropdown
  • .dropdown-menu : aplicar na div que contem os links
  • .dropdown-item : aplicar nos links do dropdown

    * button class="btn btn-info dropdown-toggle" data-toggle="dropdown" type="button" *
